# A pkg_metric subclass for general metric evaluation conditions

A pkg_metric subclass for general metric evaluation conditions

## Usage

``` r
as_pkg_metric_condition(x, ..., subclass = c())
```

## Arguments

- x:

  an object to wrap in a `pkg_metric_condition` class

- ...:

  additional arguments added as attributes to object `x`

- subclass:

  an optional subclass of `pkg_metric_condition` to include

## Value

an object after wrap `pkg_metric_condition` class.
