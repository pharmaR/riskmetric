# Helper for retrieving a list of columns which contain pkg_metric objects

Helper for retrieving a list of columns which contain pkg_metric objects

## Usage

``` r
get_assessment_columns(tbl)
```

## Arguments

- tbl:

  a [`tibble`](https://tibble.tidyverse.org/reference/tibble.html) to
  select columns among

## Value

a logical vector of `pkg_metric` column indices
